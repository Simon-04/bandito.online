Useful Commands

Code Defenition
<h1> Heading level 1, this can go down to level 6.
<p> This tag represents a paragraph.
<b> This tag makes the text bold.
<i> This tag makes the text italic
<q> This tag marks a quote in your text.
<blackquote> This tag marks a longer quote. It breaks out from the text and stands out.
<a href=""> You can use this if you want to have a have a link to a external/internal webpage.
<img src="" alt""> You can use this to upload a picture to your website.
<ul> and <ol> "<ul>" makes a bullet list while "<ol>" makes a number list.
You will need to specify the items with the tag "<li>" to list your items.
<sub> and <sup> "<sub>" makes the text move down while "<sup>" makes the text move up.
